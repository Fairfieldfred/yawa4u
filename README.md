# yawa4u

A new open source Flutter project to build a free useful multi-platform app gym routine and exercise tracking.

## Getting Started

This is a free app for tracking gym workouts. It allows the user to plan an exercise program and progress through daily workouts prechosen and based from a calandar based template. It incorporates the concept of progressive overload to help the user progress through their workouts. It also presents a growing list of exercises, workouts, training programs and lets the user create their own exercises and workouts and/or contirbute to the growing number of program templates added to the repo by users like you.
